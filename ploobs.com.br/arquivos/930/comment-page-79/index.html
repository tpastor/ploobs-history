MaxScript for games I


In this serie of tutorials i will show how to use maxscript to retrieve informations of models, materials and others informations showing how we made the ploobsexporter and how to make your own one.
Almost everything you can make in max, you can make with maxscript.

In this tutorial i will show you the pretty basic. How to create and get informations from a simple maxcript object.

The first thing you need to know is the MAXScript menu: (you see it as one of the stardard submenus)

MaxScript Menu

Basic Entries

You have basicaly 2 way to begin using maxscript

  • Text mode:
    • Begin a script in text mode, writing the logic
      • Creating a new text script file (New Script)
      • Using the listener to create objects just in time (MaxSxript Listener). (you write the command press enter and it is executed =P)
  • Visual mode:
    • Create the GUI to your script, putting buttons, text box and others common gui object (Visual MaxScript Editor …)

Language Structure

Maxscript is a language made to get your life easier. Your variables don´t have types and you don´t need “;” in the end of lines. Just like another script language, you just code.

  • Equations:
  • Strings
  • Arrays

Creating a Simple box

To create primitives, we can use a simple interface, like using a simple function; (Experiment using these comands in the MaxScript Listender — press F11)

mybox = box()

This is like using the Standard Constructor of a class.

You can create a box also passing some parameters

mybox = box length:20 width:20 height:20

Getting informations from objects

Max keeps a list that contains everything in the scene.
You can retrieve an object and assign it to a new variable for example: (max does not use index 0, the 1 is the first element)

b = objects[1]

To manipulate and change object property its important to  know what class it is:

classof b

returns:

Box

Now that we know that the object is a “Box” we can get its properties:
showclass “box.*”
returns:
Box : GeometryClass {10,0}
  .height : float
  .length : float
  .lengthsegs : integer
  .width : float
  .widthsegs : integer
  .mapcoords : boolean
  .heightsegs : integer
