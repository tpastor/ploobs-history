SharpDx Source Code Example


Some days ago i was playing with SharpDx Lib (for those who does not know it yet, it is a managed wrapper for Directx 11, check their repo: http://code.google.com/p/sharpdx/).
I have made a simple sample that loads lots of formats of models and textures (using Assimp.net for this) and show that to the screen.
In the Future, when i have time, i plan to add some physic simulation to it. Probably using Bullet Sharp or Physic.Net wrappers. For now i will focus on porting the 3D part of Monogame =P
The Sample Code can be found here: http://code.google.com/p/sharpdx-examples/
