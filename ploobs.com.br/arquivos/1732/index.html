Code Snipped: Night Vision Shader


This post effect is pretty cool and simple to build.
We tried to achieve two things, make the green like effect and add some random noise to the final image.

Different from almost all Code Snipped, we will try to explain this effect in a simple way.

The noise is the easy part, just pass a noise texture (like this, be carefull choosing the texture, different effects can be achieved just changing it a bit) and animate its uv using the time parameter. The recoved texel is used to “disturb” the final image UV reading.

The green effect is also simple, we recover the color, amplificate the luminance in parts where it is not big enough and them modulate the color using a “green like” filter.

The shader code for all of this is:

sampler TextureSampler;

texture tex;
sampler noiseTex = sampler_state
{
   Texture = ;
   AddressU  = Wrap; AddressV = Wrap;
};

struct PS_INPUT
{
	float2 TexCoord : TEXCOORD0;
};

float elapsedTime;
float luminanceThreshold = 0.1;
float colorAmplification = 4;

float4 UnderWater( PS_INPUT Input ) : COLOR0
{ 

	float2 uv;
    uv.x = 0.4*sin(elapsedTime*50.0);
    uv.y = 0.4*cos(elapsedTime*50.0);
    float3 n = tex2D(noiseTex,
                 (Input.TexCoord*3.5) + uv).rgb;
    float3 c = tex2D(TextureSampler, Input.TexCoord + n.xy*0.005).rgb;	                       

    float lum = dot(float3(0.30, 0.59, 0.11), c);
    if (lum < luminanceThreshold)
      c *= colorAmplification; 

    float3 visionColor = float3(0.1, 0.95, 0.2);
	return  float4((c + (n*0.2)) * visionColor ,1);
}

technique NighVision
{
	pass P0
	{
		PixelShader = compile ps_2_0 NightVisionShader();
	}
}
