Releasing TelemetrySystem For The Game F1 2012


F1 2012  is one of the games i most enjoy playing. (with the steering wheel =P). Since the F1 2010, the thing that i always wanted was a telemetry system. I wanted to be able to change a single car parameter and see how it actually affects my lap.

Instead of waiting for the codemaster to provide this funcionality, i decided to implement one using the car data that F1 201x series always broadcasted via UDP sockets.

My telemetry system can plot real time graphics about almost all important values like Speed, Distance, Engine RPM, Suspension height … It also let you compare in real time these variables with those from a recorded lap.

More info (including of course the Source Code in C#) can be found here:  https://github.com/tpastor/F1_201x_TelemetrySystem

Enjoy =P
