2D Bone Animation Principles Sample in XNA


Some months ago i was studying about bone animation and i decided to make an example in xna. Here i post the source code. It was build using xna 3.1.

The sample contains:

  • Simple 2D bone animation system with interpolation
  • System to record keyframes and play them after
I used the most simple way possible to store keyframe data. The sample is not directely usefull for using in games, it is more for learning purpose.
