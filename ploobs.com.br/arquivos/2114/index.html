PloobsUpdater 1.0 Released


PloobsUpdater is a simple App that helps people Downloading new versions of PloobsEngine (Engine Dll, Demos, Documentation and Auxiliar Apps). It also notifies when new versions are avaliable.

Download Link

The usage is very simple. The basic steps are:

  • Download the PloobsUpdater installer
  • Install it =P (just a next next finish process)
  • When finished, double click on its shortcut and you will see the following link on your Windows Notify area (double click on this icon also)

Ploobs Icon

  • The following Screen will open:

PloobsUploader

  • Here you can:
  • See the current version installed on your PC and the avaliable versions on our repo,  Get a PDF explaining the News on each version,  GetSelected Version and Uninstall your current Version.
  • For more info about PloobsEngine, visit our Presentation Page

Thats All !

Any Bug or Problem, please, send an email to contato@ploobs.com.br

Enjoy.
