Integrando CUDA com C# e DirectX


Estou disponibilizando um projeto exemplo que integra a tecnologia Nvidia CUDA com DirectX.

Usei o Wrapper CUDA.NET como Wrapper C# para o Cuda.

Usei o SharpDX como Wrapper C# para o DirectX 11.

O exemplo contém uma solution com dois projetos. O primeiro tem apenas um arquivo .cu que sera compilado pela Runtime do Cuda e o segundo contem o codigo DirectX e um shader mega simples para mostrar os resultados calculados no Cuda na tela.

A integracao foi feita por VertexBuffer compartilhado, é possivel tambem compartilhar texturas entre DirectX e Cuda. Para rodar o exemplo é necessario ter instalado o Cuda SDK da NVidia. As Dlls do SharpDx e do Cuda.net ja estao incluidas.

Tambem estou disponibilizando um exemplo de integracao (nao fui eu quem fiz) de Cuda com SlimDx (outro famoso wrapper C# para DirectX). Sofri para encontra-lo na internet.

Como curiosidade, eu aprendi Cuda usando o excelente livro Cuda By Examples, recomendo para todos.

A Cuda SDK dispõe de um exemplo de integração com DirectX nativo (usando C), e o livro citado mostra como integrar com OpenGL
