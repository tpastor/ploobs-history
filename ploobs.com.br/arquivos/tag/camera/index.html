Making a Simple FPS Camera in XNA 4.0

Posted by tpastor in XNA on 5 de agosto de 2011

This post will show the code of a free (no constraints in the moviment) FPS camera for XNA 4.0 It can be used as a starting point for everyone that wants to build a game that uses FPS camera style behavior.

The general ideia is pretty simple. We started creating a perspective projection matrix using the xna standard helper function CreatePerspectiveFieldOfView (fov, far plane, near plane ….) and store it a constant for all frames.
