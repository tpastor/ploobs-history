Weak Reference in C#


We know that C# is a garbage-collected language. This usually means that we (programmers) don’t need to explicitly free our own memory. When we talk about a reference to an object in .NET (and in most other garbage collected languages), we are talking about a “strong” reference. As long as that reference exists, the garbage collector won’t collect the object behind it.
A weak reference is a special type of reference where the garbage collector can still collect the underlying object, even though you still technically have a reference to it.

The common scenarios where weak reference can be used are:

  • Cache Implementation
  • Heavy (memory consumption) Objects that are rarelly used
  • Where you want to have a way to reference, but you wont hold a “Strong” reference to it 

When i was implementing ploobsengine, i fall in the following situation:
I wanted to have a global way to access objects created by the users, but i want the user to control exactely when the object will be ellected to be garbage collected (my reference could not prevent the GC to collect the object). To solve this, i created a WeakList (same as classic List but the values are weak references).

In .Net, there is a “wrapper” that can be used to create a weak reference ( very well explained here http://www.dotnetperls.com/weakreference). The problem is that this class is not “Strong typed”  (we need to convert from .Net Object class every time), so i built a simple wrapper to fix this, the code is shown below:

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
#if WINDOWS
using System.Runtime.Serialization;
#endif
namespace PloobsEngine.Utils
{


        
    /// 
    /// Weak reference Strong type
    /// 
    /// 
#if WINDOWS
    [Serializable]
#endif
    public class WeakReference : System.WeakReference
    {
        /// 
        /// Initializes a new instance of the  class.
        /// 
        /// The target.
        public WeakReference(T target)
            : base(target)
        {
        }

        /// 
        /// Initializes a new instance of the  class.
        /// 
        /// The target.
        /// if set to true [track resurrection].
        public WeakReference(T target, bool trackResurrection)
            : base(target, trackResurrection)
        {
        }
#if WINDOWS
        /// 
        /// Initializes a new instance of the  class.
        /// 
        /// An object that holds all the data needed to serialize or deserialize the current  object.
        /// (Reserved) Describes the source and destination of the serialized stream specified by .
        /// 
        ///    is null. 
        public WeakReference(SerializationInfo info, StreamingContext context)
            : base(info, context)
        {
        }
#endif
        /// 
        /// Gets or sets the object (the target) referenced by the current  object.
        /// 
        /// null if the object referenced by the current  object has been garbage collected; otherwise, a reference to the object referenced by the current  object.
        ///   
        /// The reference to the target object is invalid. This exception can be thrown while setting this property if the value is a null reference or if the object has been finalized during the set operation.
        public new T Target
        {
            get
            {
                return (T)base.Target;
            }
        }
    }
}

I also added the code of the WeakList

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;

namespace PloobsEngine.Utils.WeakReference
{
    /// 
    /// List with weak references
    /// 
    /// 
    public class WeakList : IList
    {
        private List> _innerList = new List>();

        #region IList Members

        /// 
        /// Indexes the of.
        /// 
        /// The item.
        /// 
        public int IndexOf(T item)
        {
            return _innerList.Select(wr => wr.Target).IndexOf(item);
        }

        /// 
        /// Inserts the specified index.
        /// 
        /// The index.
        /// The item.
        public void Insert(int index, T item)
        {
            _innerList.Insert(index, new WeakReference(item));
        }

        /// 
        /// Removes at.
        /// 
        /// The index.
        public void RemoveAt(int index)
        {
            _innerList.RemoveAt(index);
        }

        /// 
        /// Gets or sets the  at the specified index.
        /// 
        public T this[int index]
        {
            get
            {
                return _innerList[index].Target;
            }
            set
            {
                _innerList[index] = new WeakReference(value);
            }
        }

        #endregion

        #region ICollection Members

        /// 
        /// Adds the specified item.
        /// 
        /// The item.
        public void Add(T item)
        {
            _innerList.Add(new WeakReference(item));
        }

        /// 
        /// Clears this instance.
        /// 
        public void Clear()
        {
            _innerList.Clear();
        }

        /// 
        /// Determines whether [contains] [the specified item].
        /// 
        /// The item.
        /// 
        ///   true if [contains] [the specified item]; otherwise, false.
        /// 
        public bool Contains(T item)
        {
            return _innerList.Any(wr => object.Equals(wr.Target, item));
        }

        /// 
        /// Copies to.
        /// 
        /// The array.
        /// Index of the array.
        public void CopyTo(T[] array, int arrayIndex)
        {
            _innerList.Select(wr => wr.Target).CopyTo(array, arrayIndex);
        }

        /// 
        /// Gets the count.
        /// 
        public int Count
        {
            get { return _innerList.Count; }
        }

        /// 
        /// Gets a value indicating whether this instance is read only.
        /// 
        /// 
        /// 	true if this instance is read only; otherwise, false.
        /// 
        public bool IsReadOnly
        {
            get { return false; }
        }

        /// 
        /// Removes the specified item.
        /// 
        /// The item.
        /// 
        public bool Remove(T item)
        {
            int index = IndexOf(item);
            if (index > -1)
            {
                RemoveAt(index);
                return true;
            }
            return false;
        }

        #endregion

        #region IEnumerable Members

        /// 
        /// Gets the enumerator.
        /// 
        /// 
        public IEnumerator GetEnumerator()
        {
            return _innerList.Select(x => x.Target).GetEnumerator();
        }

        #endregion

        #region IEnumerable Members

        /// 
        /// Returns an enumerator that iterates through a collection.
        /// 
        /// 
        /// An  object that can be used to iterate through the collection.
        /// 
        System.Collections.IEnumerator System.Collections.IEnumerable.GetEnumerator()
        {
            return this.GetEnumerator();
        }

        #endregion

        /// 
        /// Purges this instance.
        /// 
        public void Purge()
        {
            _innerList.RemoveAll(wr => !wr.IsAlive);
        }
    }


}

This concept is very simple but very usefull when you are concerned with memory usage and object reference control.
