PloobsEngine Features

The following list shows some of the Main Features of the PloobsEngine 0.3 Beta (Last Released Version):
  • PC and Windows Phone 7 Support
  • Forward Rendering (PC and Windows Phone 7) and Deferred Rendering (Only for PC)
  • Artificial Intelligence (PathFinding, NN, GA, Steering Behaviors, BlackBoard, RVO Crowed Simulation …)
  • 3D and 2D Sound
  • Physics Integrated (Bepu for 3D and Farseer for 2D)
  • Nvidia Physx Support (Including Cloths and Fluids !!!)
  • 3D Animation Integrated  (Bone Animation)
  • 2D Sprite System
  • 3D Dynamic Water and Ocean with waves
  • 3D  Dynamic Reflection and Refraction
  • 3D Dynamic Lights (Spot, Point and Directional)
  • 2D Dynamic Lights
  • Support 3D Bump Mapping
  • Support 3D Specular Maps
  • Support 3D  Glow Maps
  • Support 3D Paralax Mapping
  • Dynamic Cube Map Reflection
  • Occlusion Culling
  • 3D Vegetation (Real Modeled Trees and Procedurally Generated Billboards)
  • Terrain with HeightMaps and Multitexture (Allow Giants terrain to be built)
  • Geoclipmap Terrain Support
  • 3D Dynamic Shadow (Shadow Mapping and Cascade Shadow Mapping) with filtering
  • 2D Dynamic Shadow
  • Deferred with Antialiasing (PostEffect or MLAA)
  • Post Effects: ToonShading, Blur, Noise, Wiggle, Circular Glow, DOF, Bloom, HDR, Radial Blur, Negative, Black and White, SSAO, Color Correction, Gama Correction, Saturation, Contrast, Fog, Ambient Scattering and much more …
  • 2D and 3D Extensible Particle System
  • Supports Lots of 3D Debug Primitives
  • Text Rendering
  • Transparency (with Deferred Shading, extra Forward pass)
  • EnvironmentMapping
  • Skybox
  • Billboards (Gpu Spherical and Cylindrical)
  • Animated Billboards and Textures
  • Video Player Embedded
  • Hardware Instancing (static and dynamic)
  • Advanced GUI System (using custim implementation of Neoforce)
  • Trigger System
  • Message System
  • Picking System
  • Input Control System
  • Resource Management System
  • Screen Manager System
  • Culling System (Dynamic Octree and others)
  • Cameras (First Person, Quake like, Third Person, Static, Follow Path, Interpolators …)
  • Build in Multi Threading support (Task API for the end user)
  • Engine Update runs in Multiple Threads
  • Integrated With .NET WPF
  • Integrated With 3DS Max using an Exporter Plugin
  • Scene Loader supporting Lights, Cameras, Models and Dummies.
  • Procedural Textures and Models Facilities
  • Has Lots of Math and Physic Helpers
  • Extensible Design and Easy to Use
